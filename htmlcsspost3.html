8 types of shadow effect.

HTML CSS/09-2021/8-types-of-shadow-effect

We will create these 8 types of shadow effect:-





Every single box has different type of shadow, you can use them in background of Images, CSS cards, buttons and etc. This make a webpage look much better. And every box has its own .before and .after effects you can adjust colour of shadow. 

There is a parent class which is "box" and 8 other child class which means if you want to copy only one effect then do make sure you have acknowledge parent class.

Here is our full code
